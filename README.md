# openIMIS Frontend Merankabandi module

This repository contains the openIMIS Frontend Merankabandi module files, providing Burundi-specific customizations for the Merankabandi social protection program.

The module includes M&E dashboards, payment request workflow, grievance picker overrides, beneficiary photos, and benefit plan province management.

The module uses Rollup as its build tool. Developers can leverage `yarn link` and `yarn start` during development to monitor file changes and automatically refresh the development server.
